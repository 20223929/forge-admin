## 一、它是什么？

**Forge AI 数据可视化大屏平台** 是一个基于 **Vue3 + TypeScript + ECharts + VChart** 技术栈构建的低代码可视化开发平台，专为数据大屏场景设计。

简单来说，它让你像搭积木一样设计数据大屏，更支持通过 **AI 自然语言交互**直接生成完整的大屏页面。

**一句话概括：你描述需求，AI 帮你画大屏！**

---

## 二、三大核心亮点

### 🤖 亮点一：AI 一键生成大屏

这是最令人兴奋的功能！在编辑器左侧的 AI 助手面板，你只需输入需求描述，比如：

> "生成一个电商销售数据监控大屏，包含月度销售趋势、品类占比、地区分布"

AI 就会自动分析你的需求，生成包含图表组件、布局配置、数据结构的完整大屏方案，一键应用到画布即可！

系统还内置了 4 个快捷提示词模板，覆盖最常见场景：
- 📊 电商销售数据监控大屏
- 🏙️ 智慧城市运营中心大屏
- 🏭 工厂生产数据监控大屏
- 💰 财务数据分析大屏

### 🔌 亮点二：多供应商 AI 架构

不同于绑定单一 AI 服务的方案，我们设计了灵活的多供应商架构：

| 支持的 AI 供应商 | 说明 |
|-----------------|------|
| 阿里百炼 | 接入通义千问系列模型 |
| OpenAI | 支持 GPT 系列模型 |
| 智谱 AI | 接入 GLM 系列模型 |
| Moonshot | 接入 Kimi 系列模型 |
| DeepSeek | 接入 DeepSeek 系列模型 |
| Ollama | 支持本地部署的开源模型 |
| 自定义 | 兼容 OpenAI API 格式的任意服务 |

用户可以在系统中自行配置和管理 AI 供应商，实时切换模型，调节温度参数，灵活控制生成效果。

### 📡 亮点三：对接真实后台接口

原版 GoView 主要面向纯前端使用场景，数据大多通过 Mock 或静态 JSON 配置。而 Forge 版本进行了全面改造：

- ✅ **统一认证体系**：大屏平台与 ForgeAdmin 后台管理系统共享 Sa-Token 认证
- ✅ **项目持久化**：大屏项目数据存储到后端数据库，支持多端同步
- ✅ **动态数据源**：支持配置真实的后端 API 接口，图表数据实时刷新
- ✅ **AI 服务后端化**：API Key 安全存储在后端，保障密钥安全

## 六、快速上手

### 环境要求

- Node.js >= 18.x
- Java >= 17（后端服务）
- MySQL >= 8.0
- Redis >= 6.0

### 启动步骤

```bash
# 1. 克隆项目
git clone https://gitee.com/ForgeLab/forge-admin.git

# 2. 进入大屏前端目录
cd forge-report-ui

# 3. 安装依赖
pnpm install

# 4. 启动开发服务
pnpm dev
```

### AI 功能配置

1. 登录系统后，进入 **项目 → AI 供应商** 页面
2. 新增供应商，选择预设模板（如阿里百炼）
3. 填写 API Key，点击测试连接
4. 设为默认供应商，返回编辑器即可使用 AI 生成功能
